A hands-on workshop building a production-ready research agent using the ReAct pattern with Tavily for web search, LangGraph for reasoning and tool-use, and Flyte 2 for durable orchestration. Attendees leave with a working agent, a Gradio interface, and deployable code for local or production Flyte clusters.
