A hands-on workshop covering transformer-based object detection. Participants fine-tune a DETR model on a custom dataset using Hugging Face Transformers, orchestrate the pipeline with Flyte 2, and deploy with a simple UI. Attendees leave with a fine-tuned model and a reusable pipeline applicable to their own detection challenges.
