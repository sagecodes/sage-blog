A hands-on workshop on parameter-efficient fine-tuning. Participants fine-tune an open-weight LLM with LoRA and QLoRA on a custom dataset using Hugging Face TRL, orchestrate the pipeline with Flyte 2, and deploy the model behind a simple UI. Attendees leave with a trained model, reusable pipeline code, and a portfolio-ready project.
